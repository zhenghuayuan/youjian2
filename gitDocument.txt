常用命令：

git init (初始化一个git仓库)
git add 文件名.后缀 (提交) 
git commit -m 备注 (确定提交)
git status (查看当前状态)
git diff 文件名.后缀 (查看哪些内容被修改了)
git log (查看提交日志，人物、时间)
git checkout -- 文件名.后缀 (撤销git add的某个文件)
git rm 文件名.后缀 ( 'rm a.txt(或手动删除)' => 'git rm a.txt' => 'git commit -m') (文件夹 -r)
git checkout -- 文件名.后缀 (恢复某个版本)
git push origin master (确认提交到github上)
git clone https://github.com@github.com:zhenghuayuan/项目名字.git (github克隆 创建一个文件夹，进去)
